# Nginx Configuration on Windows with USocketNet

USocketNet is a self-host Realtime Multiplayer Server for your Games or Chat application.

## Getting Started

This is a official nginx configuration by USocketNet in a Windows Environment.

### Prerequisites

Unused port for 19090~2, 6060~2, 9090~2, 8080~2.

## Running the tests

To check if the nginx is already running using this project just double clicked on "_check.bat". It will show taskid of nginx instance, usually, it will create 3 instance. Dont worry when you see 3 instance of nginx.

## Deployment

What you need is a windows operating system then clone this project
